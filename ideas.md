# Eshop 前端設計規劃

## 架構概覽

Eshop 版面遵循以下層級結構：
```
所有 (Home)
  ├─ 系列 (Series)
  │   ├─ 專輯 (Albums)
  │   │   ├─ 實體商品 (Physical)
  │   │   ├─ PDF 商品 (Digital - PDF)
  │   │   └─ MMO 商品 (Digital - MMO)
  │   └─ ...
  └─ ...
```

## 設計方向

### 設計理念 1: 極簡未來主義
- **設計運動**: Minimalism + Futurism
- **核心原則**: 
  - 清晰的層級結構，導航直觀
  - 大量留白，強調內容
  - 單色調主體 + 霓虹色點綴
  - 無縫過渡，流暢動畫
- **色彩哲學**: 深灰/黑色背景 + 霓虹藍/紫色強調，營造科技感與未來感
- **版面範式**: 垂直滾動為主，側邊欄分類，卡片網格展示商品
- **簽名元素**: 
  - 細線分割符
  - 動態漸變邊框
  - 懸浮效果卡片
- **互動哲學**: 滑過卡片時微妙浮起，點擊時發光反饋
- **動畫**: 平滑淡入淡出，卡片進場時的輕微縮放，懸浮時的發光效果
- **排版系統**: 現代無襯線字體（Poppins/Outfit）+ 高對比度，強調數字與科技感

**概率**: 0.08

### 設計理念 2: 手工藝溫暖風格
- **設計運動**: Artisanal + Warm Minimalism
- **核心原則**:
  - 自然色調，溫暖親切
  - 手繪感插圖與紋理
  - 非對稱版面，充滿個性
  - 強調故事性與人文連結
- **色彩哲學**: 米色/奶油色背景 + 土色/暖棕色強調，營造手工藝的溫暖感
- **版面範式**: 不規則網格，左右交替排列，融入插圖與文字敘述
- **簽名元素**:
  - 手繪邊框
  - 紙質紋理背景
  - 有機形狀卡片
- **互動哲學**: 溫和的懸浮效果，點擊時的輕微旋轉或傾斜
- **動畫**: 柔和的進場動畫，卡片輕微搖晃，頁面轉換時的翻頁效果
- **排版系統**: 混合襯線與無襯線字體（Georgia + Poppins），營造溫暖與現代的平衡

**概率**: 0.07

### 設計理念 3: 大膽藝術裝飾風格
- **設計運動**: Art Deco + Contemporary Bold
- **核心原則**:
  - 幾何圖案與對稱性
  - 大膽色彩碰撞
  - 豐富的視覺層次
  - 奢華感與精緻度
- **色彩哲學**: 深金色/黑色背景 + 翡翠綠/珊瑚紅/金色強調，營造高端與藝術感
- **版面範式**: 對稱中軸線，卡片排列成網格，邊框與裝飾線條豐富
- **簽名元素**:
  - 金色分割線
  - 幾何圖案背景
  - 邊框裝飾
- **互動哲學**: 點擊時的華麗動畫，懸浮時的色彩變化
- **動畫**: 進場時的旋轉與縮放，點擊時的脈衝效果，頁面轉換時的優雅淡出
- **排版系統**: 高對比度的現代字體（Playfair Display + Montserrat），強調奢華感

**概率**: 0.06

---

## 選定設計方向

**已選定：極簡未來主義**

此設計方向最適合 Eshop 的多層級導航結構，提供清晰的視覺層級與直觀的用戶體驗。霓虹色點綴能突出商品，而流暢的動畫增強交互感。

## 實現細節

### 色彩系統
- **背景**: `#0f0f0f`（深灰黑）
- **卡片背景**: `#1a1a1a`（稍淺灰黑）
- **主要強調色**: `#00d9ff`（霓虹藍）
- **次要強調色**: `#a855f7`（紫色）
- **文字主色**: `#ffffff`（白色）
- **文字輔色**: `#a0a0a0`（淺灰）

### 排版
- **顯示字體**: Poppins Bold 700（系列、專輯標題）
- **正文字體**: Poppins Regular 400（描述、商品名稱）
- **強調字體**: Poppins SemiBold 600（分類標籤、價格）

### 互動效果
- 卡片懸浮時：輕微上升 + 邊框發光
- 點擊商品時：發光脈衝 + 平滑過渡到詳情頁
- 頁面轉換：淡入淡出 + 0.3s 過渡時間

---

## 頁面結構

1. **首頁 (Home)**: 系列總覽卡片網格
2. **系列頁 (Series)**: 該系列下的所有專輯
3. **專輯頁 (Album)**: 該專輯下的商品分類（實體/PDF/MMO）
4. **商品列表頁 (Products)**: 按類型篩選的商品列表
5. **商品詳情頁 (Product Detail)**: 單個商品的詳細信息與購買選項
